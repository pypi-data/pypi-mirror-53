ndn-python: A Named Data Networking client library with AsyncIO support in Python 3
===================================================================================

Instructions for developer
--------------------------

To setup python3.6 virtual environment:
```bash
python3.6 -m venv venv
. venv/bin/activate
pip3 install -e .
pip3 install pytest pytest-cov
```

To do unit tests:
```bash
make test
```
