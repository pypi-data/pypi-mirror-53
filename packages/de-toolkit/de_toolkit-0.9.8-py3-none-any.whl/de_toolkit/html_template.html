Base:
Number of Columns Number of Rows
$num_cols $num_rows

Coldist:
The coldist boxplot shows the distribution of the counts in each column of the data
matrix. Hover over an outlier point to see which row it came from,or hover over the
boxplot to see the first and third quartile values, the median, and the cap values. $coldist_boxplot

Colzero:
The colzero output is a bar chart depicting the zero fractions for each sample. Hover
over each bar to see the value of the zero fraction for that sample. $colzero

Rowzero:
The first rowzero plot is a scatterplot of the zero fractions against nonzero means for
each row. The second rowzero plot is a scatterplot of the zero fractions against the
overall mean of the row. For each of the scatterplots, hover over each point to see
which row the point comes from, as well as the x and y values of the point. The third
rowzero plot is a histogram of the zero fractions. Hover over the bar to see the
frequency of the bin. $rowzero_scatter
$rowzero_scatter2
$rowzero_hist

Entropy:
The entropy plot is a histogram of the entropies of each row. Hover over the bars to
see the frequency of the bin. $entropy
